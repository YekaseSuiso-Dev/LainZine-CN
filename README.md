# 雪莉的LainZine中文版
这里是LainZine的中文翻译版，仅供参考...雪莉以前和原团队联系过以推进LainZine的中文化，因为恶俗wiki对我们的迫害，项目被搁置了。
## so what is this thing?
Lainzine 是一本免费、非营利性质的电子杂志[（zine）](https://en.wikipedia.org/wiki/Zine)，主要由[《Serial Experiments Lain》](https://zh.wikipedia.org/zh-hk/%E7%8E%B2%E9%9F%B3)（90年代末的经典动漫）的粉丝创建，也献给这些粉丝，以及这部作品在过去几十年里所激发出的众多相关社区、网站和尘封的艺术项目。
杂志涵盖的主题包括“数字生活”、艺术与计算机的交汇、[赛博朋克](https://en.wikipedia.org/wiki/Cyberpunk)主题、编程与操作安全（opsec）、伪宗教式的技术胡言乱语，以及人们想分享的任何其他内容，所有内容都以受 Lain 启发的 ａｅｓｔｈｅｔｉｑｕｅ（赛博美学）风格呈现。
